# 宝宝英语乐园

一个适合 3 岁左右孩子使用的英语启蒙网页。

## 功能

- 500+ 基础词汇
- 大按钮点读
- 看一看模式
- 找一找模式
- 本地自定义词库
- 可直接放到 GitHub Pages

## 怎么分享

1. 把这个文件夹上传到 GitHub 仓库。
2. 在仓库设置里打开 GitHub Pages。
3. 选择根目录发布，保存后就能分享链接。

## 怎么继续加词

- 直接编辑 `data/vocabulary.js`
- 或者在网页里的“家长更新区”先加词，再导出备份

## 本地打开

如果你想先在本机看效果，直接用任意静态服务器打开这个目录即可。

## 备注

这个版本故意做得比较简单，方便孩子直接点、直接听、直接玩。
